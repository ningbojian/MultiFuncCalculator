# 多功能计算器应用

界面简洁，无广告，适合工作学习间隙放松。

## 主要功能

### 基本计算器
*   加减乘除运算
*   基本数学函数：平方、倒数、平方根(√x)

### 科学计算功能
*   三角函数：sin()、cos()、tan()、sec()、csc()
*   角度转换：rad()
*   指数运算：x^y
*   阶乘：x!
*   对数：ln()、lg()
*   常数：e、π
*   清除功能：AC、←

### 附加功能
*   进制转换（10 ↔ 2 ↔ 16 ↔ 8）
*   斐波那契数列计算

### 游戏功能
*   数字华容道/八数码
*   9x9数独
*   猜数字小游戏
*   测反应速度
*   2048游戏

## 项目特色
*   🌿 画面极简，清新无广
*   ⚡ 功能齐全，摸鱼必备
*   🎮 多种睿智小游戏集合
